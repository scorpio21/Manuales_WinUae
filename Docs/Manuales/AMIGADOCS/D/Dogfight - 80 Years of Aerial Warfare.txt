Short key guide for DOGFIGHT.
-----------------------------
FIGHTER Controls
Mission timer on/off
+            Increase Power
-            Decrease power
[shift] +    Maximum power
[shift] -    Maximum power decrease
]            flaps down  

UP ARROW     Move UP     |
DOWN ARROW   Move DOWN   |
LEFT ARROW   Move LEFT   |
RIGHT ARROW  Move RIGHT  | or usual JOYSTICK Movements.

<            Left rudder
>            Right rudder
G            Landing gear up/down
W            Wheels brakes on/off
B            Air brakes on/off
R            Radar range
A            Resource menu

[shift] Z    Accelerate time
[shift] X    Return to normal time
[CTRL]  T    Mission timer on/off
[CTRL]  E    Bail out
        L    Launch from carrier

HARRIER only Controls
---------------------
1            Nozzles to Horizontal
2            Nozzles to 45 degrees
3            Nozzles to Vertical

WEAPON control
--------------
(space bar)  Fire Weapon [or press j/stick fire button]
(return)     Change weapon

DECOY Control
-------------
C             Chaff release
F             Flare release

Target control
--------------
T             Select new target
[shift] T     Return to player aircraft
Y             Select nearest enemy
U             Select Nearest ground target
[ALT]   B     Auto intercept mode

VIEW keys
---------
F1............Forward cockpit view
F2...........Instrument view/press S for half forward
F3...........Tactical view
F4...........Inverse Tactical view
F5...........Heads up view
F6...........Left view
F7...........Right view
F8...........Rear view
F9...........Map screen/zoom with keypad + and -
F0...........External view

SHIFT+F1.... slot view

VIEW controls
-------------
9............Pitch view up
3............Pitch view down
0............Rotate view left
. ...........Rotate view left
5............Centre view
+............Zoom in
-............Zoom out

Mouse Controls
--------------
left mouse   Change target on F9 MAP VIEW
Right mouse  Centre selected object on F9 MAP VIEW

Game options
------------
[ALT] D      Detail level
[ALT] C      Shape complexity
[ALT] V      Sound mode
      P      Pause game
ESCAPE       Quit game
