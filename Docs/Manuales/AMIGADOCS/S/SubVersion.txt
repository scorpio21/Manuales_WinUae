                              INSTALLATION


SubVersion can be played from a floppy disk or it can be installed on your
hard drive.

To play SubVersion from a floppy disk:

   Insert the floppy disk into the drive.
   Double-click on the disk icon.
   Double-click on the SubVersion icon.
   Enjoy the game.

To play SubVersion from a hard drive:

   Insert the floppy disk into the drive.
   Drag the SubVersion disk icon to the hard drive icon.
      The system will create a drawer called SubVersion and will copy
      all the files for the game to this drawer.
   Double-click on the hard drive icon.
   Double-click on the SubVersion drawer.
   Double-click on the SubVersion icon to start the game.

CONTROL PANELS

The game controls are the same as listed in the manual.  The only 
differences in their operation are keyboard equivalents.  Each command
can now be executed by using the underlined character within that control
panel.  The button clusters and slider bars are operated by using the
keypad and the arrow keys.  Illustrations showing these functions are
below.

Main Menu Keyboard Shortcuts            Auxilliary Controls Shortcuts
 
T - Torpedo                             S - Surrender
H - HMS                                 D - Self-Destruct
M - Missile                             A - Abandon Ship
C - Crew                                Q - Quit SubVersion
R - Resupply                            N - Start New Mission
D - Depth                               R - Rig for Silent Running
P - Power                               C - Cancel
E - Engine
A - Auxilliary Controls

Subsystem Panel Shortcuts

Button Cluster - The numbers on the keypad correspond to the direction
                 on the button cluster.  For example, pressing 4 on the
                 keypad will set the direction LEFT.

Execute - Press "Return" or the underlined character.

Abort/Cancel - Press the underlined character or the "Escape" key.

Slider Bars - Controlled with the up/down arrows.  To move in larger
              increments, hold down the shift key.
