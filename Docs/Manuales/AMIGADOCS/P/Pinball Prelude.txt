Joystick Info:
--------------

Start game:    Fire during demo/credits.
Release ball:  Fire and pull down.
Left flipper:  Pull left.
Right flipper: Pull right.
Both flippers: Pull down.
Tilt left:     Fire and pull left.
Tilt right:    Fire and pull right.

Keyboard Info:
--------------

Start game:    Enter/space.
Abort game:    Escape key.
Replay:        `R' key.
Release ball:  Space or down arrow key.
Left flipper:  Left shift key.
Right flipper: Right shift key.
Tilt left:     Left alt key.
Tilt right:    Right alt key.
