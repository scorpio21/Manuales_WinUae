               SHORT DOCS FOR ACTION CAT *AGA* / 1001 SOFTWARE
               -----------------------------------------------

ACTION CAT IS A JUMP AND RUN GAME FOR ONE PLAYER ONLY!
THE AIM OF THE GAME IS TO FIND THE EXIT OF THE LEVEL AND KILL AS MANY
ENEMYS AS POSSIBLE! FINALLY IT`S YOUR ORDER TO DESTROY THE TOOTHPASTE
FACTORY!
EVERYTIME YOU KILL A ROBOTENEMY YOU CAN
COLLECT A CRYSTAL WHICH CAN BE SWAPPED LATER TO IMPORTANT THINGS!
TO REACH THE END OF EACH LEVEL YOU POSSIBLY HAVE TO OPEN SEVERAL DOORS,
THIS CAN ONLY BE DONE BY TURNING THE SWITCHES!
(JOYSTICK UP + FIRE BUTTON!) SOMETIMES YOU CAN TURN THE SWITCHES AS
MUCH AS YOU WANT THE DOOR WONT OPEN ANYWAY! IN THIS CASE YOU HAVE TO
COLLECT THE RIGHT KEY TO OPEN THE DOOR!

KEYS IN MAINMENU!
-----------------

S  - VIEW HIGHSCORE LIST!
F1 - FLIP TO OPTIONS MENU

INGAME-KEYS:
------------

SPACE - ACTIVATE NOVABOMB
AMIGA - ACTIVATE POWERSHOOT
ALT   - ACTIVATE ROCKET
RETURN- OPEN INVENTORY MENU (FOR KEYS!)
HELP  - PAUSE
ESC   - GUESS WHAT!?

GAME NOTE:
----------

THE GAME WORKS ON THE AMIGA 1200 AND SHOULD DO SO ON THE AMIGA 4000 ALSO!
BUT IT IS -NOT- TESTED THERE!

TIPS:
-----

YOU SHOULD COLLECT AS MANY POWERUPS AS POSSIBLE , THEY WILL
GIVE YOU MORE ENERGY OR AN EXTRA WEAPON!

EVERY 100000 POINTS YOU GET AN EXTRA LIVE!

WATCH OUT FOR SECRET ROOMS! AT THE GROUND FOR EXAMPLE!

SWICHTES CAN NOT ONLY OPEN DOORS IT CAN ALSO MOVE WALLS!

SPECIAL OPTIONS DURING PLAYING:
-------------------------------

JOYSTICK DOWN + FIRE BUTTON - SET BOMB

JOYSTICK DOWN - ACTIVATE TELEPORT! 
(THE TELEPORTER CAN ONLY BE ACTIVATED IF YOU ARE AT THE TELEPORTERPLACE!)

TO SKIP THE INTRO SEQUENCE SIMPLY LOAD THE GAME FROM WORKBENCH!
