BREF HISTORIQUE
Il y a bien longtemps, dans le petit village d'ISSEGEAC en plein coeur
du Périgord Noir, on constatait des phénomènes bizarres et de bien 
étranges disparitions. Les habitants d'Isségeac vivaient dans la
terreur. L'angoisse grandissait de jour en jour et le doute s'installait.
La population fit alors appel au grand Homorius, exorciste de renom.
Après maintes incantations tirées de son grimoire, le calme revint.
Avant de partir, il dit aux villageois :
"Braves gens, la pais est maintenant revenue. N'ayez plus peur ! Je
dois m'en aller mais je vous laisse ce grimoire. Il sera votre protection,
ne l'égarez jamais. "
Sur ces dernières paroles, il reprit la route et le grimoire fut
confié au patriarche du village qui n'était autre que votre aieul.
Pendant bien des années, Isségeac vécut dans la sérénité, mais...
Un soir, on vit arriver au village un étranger.
Nul ne distinguait son visage. Sa démarche était lourde. Soudain, il 
s'arreta et leva les bras. Terrorisés, les habitants se réfugièrent
dans leurs maisons. Les volets se fermèrent rapidement.
La nuit tomba. Un cri déchirant retentit...
C'est au petit matin qu'il fallut se rendre à l'évidence :
Le corps de votre aieul gisait, inerte et le grimoire avait disparu...

LE JEU

BLACK SECT se joue de préférence à la souris.
Toutefois, le clavier peut entièrement se substituer à la souris
sur PC et pour la plupart des actions sur ST et AMIGA.
Pour choisir parmi les diverses options, positionnez la souris sur
l'icone de votre choix puis cliquez sur le bouton gauche. Au cours
du jeu, vous pouvez répéter la dernière action ou le dernier
déplacement choisi en cliquant sur le bouton de droite de la souris.

A: DEPLACEMENTS :
Pour accéder au menu déplacements. Seules les directions affichées
à l'écran sont accessibles.
B: INVENTAIRE :
C'est la liste des objets que vous avez sur vous. L'objet que vous avez
en main est présenté en haut de l'inventaire. Pour prendre un autre
objet de l'inventaire, cliquez sur celui de votre choix : il s'affichera
alors en haut à la place de l'autre.
Nota : si vous aviez déjà un objet en main, il prendra la place de celui
que vous venez de sélectionner.
Vous pouvez avoir, au maximum, 5 objets sur vous (y compris celui en main).
Toutefois, votre force étant limitée, le nombre d'objets possédés sera 
fonction de leurs poids et de leurs volumes.
C: SAUVEGARDE :
5 sauvegardes sont possibles. Celles déjà utilisées sont marquées d'un
point. Vous pouvez, bien sur, écraser une ancienne sauvegarde pour
sauver un nouvel état. Nota: les chargements ne se font qu'en début de
partie. C'est volontaire : pour vous forcer à ne pas jouer avec le temps
...comme dans la vie réelle !
D) CLAP ACTION :
16 actions sont possibles : du haut à gauche en bas à droite :
REGARDER - PRENDRE - DEPOSER - SOULEVER - APPUYER - BRISER - OUVRIR -
FERMER - TOURNER - PARLER - MONTRER OU DONNER - UTILISER - FOUILLER -
TIRER - METTRE - SE CACHER...
Le clap est divisé en deux : cliquez sur la partie supérieure : les
icones de chaque action s'affichent à l'écran.
cliquez sur la partie inférieure : un petit damier apparait.
Chaque zone correspond à une action, sans la nommer. L'ordre des
actions est respecté. Optez pour cette formule pour un gain de temps.
LES ACTIONS PARTICULIERES :
Pour effectuer certaines actions, vous devez absolument avoir un objet
en main :
-"montrer/donner": présentez l'objet à un personnage.
-"mettre": désignez à l'écran l'endroit choisi.
-"utiliser": cliquez sur la zone désirée pour l'utilisation de l'objet.
-"déposer": l'objet est désormais dans le tiroir (MODE OBJET:ZONE2).
E) BOUEE:
Vous apporte une aide. Une indication est donnée pour chaque lieu en
mode LIEUX et pour chaque objet en mode OBJET.
1) LANKHOR :
Pour connaitre le "pourcentage d'aventure résolu". Ce pourcentage est
remis à jour toutes les heures.
2) HEURE/JOUR :
Vous disposez de 3 jours pour mener à bien votre quete.
Le temps est primordial, la date et l'heure (AM: de minuit à midi et
PM : de midi à minuit) vous sont notifiées. Nota : le temps dans le
jeu passe 5 fois plus vite que le temps réel.
Vous avez la possibilité de faire évoluer le temps en cliquant :
- sur l'heure pour faire passer les heures
- sur les minutes pour avancer de 5 minutes
- sur le jour pour faire passer 24 heures
- sur AM ou PM pour avancer de 12 heures.
Nota : des messages apparaissent à l'écran à des heures précises.
Lorsque vous faites évoluer le temps, ces messages bloquent le jeu
afin que vous puissiez les lire. Vous devez cliquer sur le bouton gauche
de votre souris pour que le temps continue à tourner.

Vous vous retrouvez dans ce mode lorsque vous voulez faire une action sur:
- 1 objet de l'inventaire (clik en zone1)
- 1 objet déposé dans un lieu (clik en zone2)
- lorsque vous fouillez une zone de l'écran contenant des objets.

A) pour sélectionner l'objet suivant
B) pour sortir de la sélection d'objets.
ZONE1) : OBJET EN MAIN :
Un clik dans cette zone permet d'agir sur les objets de l'inventaire. Ceux-ci
apparaissent en ZONE3 (qui permet l'action sur les objets). De plus, si un
objet est représenté dans cette zone, il s'agit de l'objet que vous avez
en main.
ZONE2) : DEPOT D'OBJETS :
- Rien n'est affiché : aucun objet n'a été déposé dans ce lieu.
- un tiroir : pour vous rappeler que vous avez déposé des objets sur le lieu
ou vous etes. 
- un clik sur le tiroir : les objets déposés dans le lieu apparaissent en
ZONE3.
ZONE3) : ACTIONS SUR LES OBJETS :
Cliquez sur un objet affiché dans cette zone, apparait alors le MENU 
ACTIONS SUR OBJETS :
7 actions sur objets sont possibles : 
REGARDER - PRENDRE - FOUILLER - LIRE - UTILISER - SENTIR - TOUCHER.
Seule l'action TOUCHER vous demande de cliquer sur un endroit précis
de l'objet. Les actions particulières :
- utiliser un objet sur un autre objet : l'objet sur lequel l'action
va etre faite doit etre affiché en ZONE3 et celui servant à faire
l'action en ZONE1.
- fouiller un objet contenant d'autres objets : l'objet fouillé passe
en ZONE2 comme objet de référence ; les objets découverts s'affichent
en ZONE3 pour actions éventuelles.

LE JEU AU CLAVIER
 ATARI ST ET AMIGA :

I) MODE LIEUX

LE PAVE NUMERIQUE :
Des touches du pavé numérique permettent l'accès direct aux actions du
mode LIEUX :

( : regarder  ) : prendre  / : déposer  * : soulever

7 : appuyer   8 : briser   9 : ouvrir   - : fermer

4 : tourner   5 : parler   6 : montrer/donner  + : utiliser

1 : fouiller  2 : tirer    3 : mettre   ENTER : se cacher

LES FLECHES DU CLAVIER SERVENT POUR LES DEPLACEMENTS

LES TOUCHES DE FONCTIONS :

F1 A F4 : pour sélectionner un objet de l'inventaire et le prendre en main.


II MODE OBJET

LE PAVE NUMERIQUE : 
4: REGARDER - 5: PRENDRE -6: FOUILLER - +: LIRE - 1: UTILISER - 
2: SENTIR -3: TOUCHER

GENERALITES

lorsqu'une question attendant comme réponse "oui ou non" vous est posée,
vous avez le choix entre :
- appuyer sur les touches O ouN
- cliquer à l'écran sur oui on non...

LA PROTECTION
Lorsque le programme vous demande de designer un mot, cherchez le dans le
manuel du jeu et cliquez à l'écran sur celui choisi parmi les 8 proposés.
(Vous ne disposerez pas de cette option, désolé (LAURENZO/PASSWORDS)...)

QUELQUES CONSEILS :
- Nous vous conseillons vivement de tracer un plan des lieux et de noter
tous les indices trouvés dans chaque.
- Pensez également à inscrire tous les objets déposés dans chaque lieu car
cela vous évitera des déplacements inutiles
- N'oubliez pas aussi d'effectuer des sauvegardes régulièrement : vous
n'etes pas seul et votre présence peut déranger certains !
