LOADING INSTRUCTIONS
~~~~~~~~~~~~~~~~~~~~
Insert game disk in DF0: and hit joystick fire-button on title screen.
Hit fire when prompted.

GAME SET-UP
~~~~~~~~~~~
After copyright notice, select number of overs to be played by clicking on 
the plus and minus gadgets with joystick. When finished, click on done. 
Select one ot two players by clicking on the respective gadgets with 
joystick. Next, select the batting team (CPU/Player two) by clicking on 
desired team's name with joystick. Then select bowling team (Player one) by 
clicking on desired team's name with joystick. You will then be presented 
with the following screen :

                COMPETITON INFO

Game Action Screen - Displays either bowling, batting or fielding action.

Umpire - Displays the umpire's decision on your last play.

Radar (?) - Displays your fielders' location on pitch.

[C] - Copyright notice.

Batting Screen - Displays batting positions and actions.

Batting Selection & Info - Select player to bowl next. 

Bowler Selection & Info - Select player to bat next.

Game Info - Displays score, amount of balls bowled.

Competition Info - Displays colours of teams who are playing.


TEAM SELECTION
~~~~~~~~~~~~~~
You will see the batting and selection screen flashing. Select the people you 
would like to bowl by clicking on their names with the joystick fire-button.

When you are batting, use the joystick to select the people you would like to
bat.


KEYBOARD CONTROLS
~~~~~~~~~~~~~~~~~
P - Pause

Any Keys Before Bowling - Enters game options screen (SEE BELOW).


GAME OPTIONS SCREEN
~~~~~~~~~~~~~~~~~~~
Fielding positions, changes positions of fielders on the pitch. Use < and >
gadgets to change positions. Clik on OK when finished, or CANCEL to quit.

Scoreboard, displays batting score of teams.

Retire batter, retires the current batter from the game.

Game length, alter number of overs to be played. Use < and > gadgets to 
alter.

Team Editor, This has the following controls in it:


BATTING			FIELDING                 BOWLING 
~~~~~~~                 ~~~~~~~~                 ~~~~~~~

POWER      + 0 -        SPEED      + 0 -         SPEED      + 0 -

ACCURACY   + 0 -        CATCHING   + 0 -         ACCURACY   + 0 -

                        THROWING   + 0 -

LOAD TEAM		SAVE TEAM		TEAM COLOUR
~~~~~~~~~		~~~~~~~~~		~~~~~~~~~~~


Use the + and - gadgets to change the rating being edited.

Load Game, load a saved game.

Save Game, save game to disk.

Cancel, return to game.

down while bowler runs up to crease (wicket).
