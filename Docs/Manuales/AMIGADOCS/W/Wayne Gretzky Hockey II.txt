                INSTALLING AND STARTING WAYNE GRETZKY HOCKEY 2
                ----------------------------------------------

1) IF NECESSARY, KICKSTART YOUR COMPUTER. AT THE WORKBENCH PRROMTP, INSERT
   YOUR WAYNE GRETZKY HOCKEY 2 DISKETTE INTO THE THE INTERNAL DISK DRIVE.
   IF YOUR AMIGA IS ALREADY UP AND RUNNING, INSERT YOUR WAYNE GRETZKY
   HOCKEY 2 DISKETTE AND 'WARM BOOT' BY HOLDING DOWN THE CTRL KEY BOTH 
   AMIGA KEY'S AT THE SAME TIME.(MAYBE HE MEANS RESET!!)

2) AFTER A FEW SECONDS, YOU WILL SEE AN ICON THAT SAYS' WGH2.' DOUBLE KLICK 
   ON THIS WITH THE LEFT MOUSE BUTTON.

3) THIS WILL GIVE YOU A WINDOW WITH THE FOLLOWING 2 ICON'S:

   WGH2 - DOUBLE KLICK ON THIS ICON TO RUN THE GAME FROM DISK

   INSTALL WGH2 - DOUBLE CLICK ON THIS ICON TO INSTALL THE GAME TO YOUR 
   HARD DRIVE. THE GAME WILL BE INSTALLED IN A DIRECTORY CALLED WGH2!.


  KEYBOARD
 ----------

   ALTHOUGH THE KEYBOARD PROVIDES VARIOUS GAME FUNCTIONS, SUCH AS PAUSING
   OR CHANGING THE LINES, THE KEYBOARD CANNOT ACT AS A CONTROLLER OPTIONS
   ARE MOUSE AND JOYSTICK ONLY.


  SELECT COLOR MENU
 -------------------
 
   MENU COLOR OPTIONS ARE 0-63. HELMET COLOR OPTIONS ARE 0-17.


  TEAM CONSTRUCTION MENU
 ------------------------

  EDIT ROSTER - TO CHANGE A CHARACTERISTIC, HIGHLIGHT THE ITEM TO BE CHANGED
  AND THEN HOLD DOWN THE MOUSE OR JOYSTICK BUTTON WHILE MOVING THE MOUSE OR 
  JOYSTICKIN THE DIRECTION OF THE CHANGE. LET GO OF THE MOUSE OR JOYSTICK 
  BUTTON WHEN YOU REACHTHE NEW VALUE.


  PLAY EDIT MENU
 ----------------
 
  CURRENTLY PLAY - TO CHANGE THE CURRENT PLAY,  HIGHLIGHTED THE ITEM
  TO BE CHANGED AND THEN HOLD DOWN THE MOUSE OR JOYSTICK BUTTON WHILE 
  MOVING THE MOUSE OR JOYSTICK IN THE DIRECTION OF THE CHANGE. LET GO 
  OF THE MOUSE OR JOYSTICK BUTTON WHEMN YOU REACH THE NEW VALUE.


  PASSING AND SHOOTING
 ----------------------

  YOU MAY NOT PASS AND SHOOT USING THE KEYBOARD.


  SELECTING THE PLAYER BEING CONTROLLED
 ---------------------------------------

  TO CHANGE THE PLAYERS BEING CONTROLLED, YOU MAY USE THE KEYS LISTED
  ON PAGE 36 OR YOU MAY CLICK ON THE PLAYER YOU WISH TO CONTROL.

  
  SAVE CONFIGURATION
 --------------------

  FROM THE GAME SETUP MENU SELECT THE 'GAME DOS' OPTION. SELECT 
  'SAVE CONFIGURATION'. ONCE DOEN, YOUR CURRENT CONFIGURATION OF THE 
  OPTIONS LISTED UNDER CONFIGURATION FILE WILL BE SAVED. THE KEYBOARD 
  CONFIGURATION CANNOT BE CHANGED.

  CONFIGURATION FILE
 --------------------

  IF YOU CHOOSE 'SAVE CONFIGURATION' FROM ANY DOS MENU, CERTAIN OPTIONS 
  PRESENTLY SELECTED WITHIN THE GAME SETUP MENU AND THE TEAM BEHAVIOR MENU
  WILL BE SAVED. THESE OPTIONS ARE LISTED BELOW.

  WITHIN THE GAME SETUP MENU:

  - TYPE OF PLAY OPTIONS
  - TEAM QUALITY LEVELS
  - CONTROLLER TYPES- MOUSE OR JOYSTICK
  - REFEREE OPTIONS
  - GAME TYPE
  - FIGHTS OPTION
  - CURSOR SIZE
  - PERIOD LENGTH
  - TRAILS OPTION
  - GAME SPEED

  WITHIN THE TEAM BEHAVIOR MENU:
 
  - TEAM BEHAVIOR SETTINGS FOR HOME TEAM
  - TEAM BEHAVIOR SETTINGS FOR VISITIING TEAM

  
  CONTROLLER TYPES
 ------------------
 
  JOYSTICK:

  THE 'FIRE' BUTTON IS USED DOR <SELECT>. IF YOUR JOYSTICK HAS A SECOND
  BUTTON, USE IT FOR <DEMAND>. IF YOUR JOYSTICK DOES NOT HAVE A SECOND 
  BUTTON, PLAYER 1 SHOULD USE THE LEFT ALT-KEY AND PLAYER 2 THE RIGHT
  ALT-KEY TO <DEMAND>.


  SCORE BOARD ON/OFF
 --------------------
 
  PRESS 'SPACE' TO TOGGLE THE SCOREBOARD ON/OFF
