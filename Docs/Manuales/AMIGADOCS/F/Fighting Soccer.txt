ARROW MODE
----------
If keyboard mode is selected the arrow moves around the player showing in
which direction you kick, that is, there is independant movement of the
arrow.

If joystick mode is selected, the arrow will automatically point in the
direction of the player, and therefore the ball will be kicked in that
direction.

GAME CONTROLS
-------------
ST/AMIGA

			2 = Player 1

			Player has not got ball = Slide
			Ball in the air = Header

DOUBLE FIRE		Player has ball = Long Kick
			Player has not got ball = Slide
			Ball in the air = Header
