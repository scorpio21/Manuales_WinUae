                   INSTRUCTION SHEET FOR AMIGA COMPUTERS

              Supplied by Stark and typed by Mictlantecuhtli

  We strongly recommend that you make a backup copy of your King's Bounty
disk and use the copy for game play. Refer to you AmigaDOS manual for
information regarding disk copying and backups.

512K Amiga Loading Instructions

  To run King's Bounty on a 512K Amiga, simply insert the King's Bounty
disk in DF0: and turn on your Amiga or reboot (CRTL-AMIGA-AMIGA). On a 512K
machine, King's Bounty MUST be booted.

  Note: to run King's Bounty on a 512K machine, you MUST disconnect any
external drives. Enhanced digitized sound will not play on 512K machines
(SORRY).

1 Meg (or more) Amiga Loading Instructions

  On memory-expanded Amiga's, King's Bounty may be run from the Workbench
by double clicking on its icon, or from the CLI by typing: "KB: King's
Bounty" <CR>. You may also run King's Bounty by booting with the King's
Bounty disk in DF0:. You do NOT need to disconnect external drives on an
expanded machine.

Hard Drive Installation

  King's Bounty may be installed on your hard drive by executing the
KBINSTALL script in the S directory of the King's Bounty disk. The script
will create a directory call BOUNTY on DH0: and copy the necessary files.
If you wish to install to another volume, simply edit the script, replacing
DH0: with the desired volume name. IMPORTANT--After installing to the hard
drive, you MUST add the following sequence to your startup-sequence:
Assign KB: DH0:Bounty.

Changes

  A few changes have been made on the Amiga version of King's Bounty.
Keyboard Equivalents:  CONTROL-S will save your game, 'E' will end the
current game, 'H' or the Help key will bring up the help box (described in
the OPTIONS menu in the manual). Most requesters can be exited with the ESC
key. Pressing RETURN or ENTER is equivalent to clicking on an 'OK' button.
Pressing the SPACE BAR while a sound is playing will abort that sound.

Side Icons

Selecting the CONTRACT icon (topmost) will bring up the Contract
Information Screen.
Selecting the CATAPULT icon will bring up the View Army Screen.
Selecting the MAGIC PENTAGRAM will bring up the Use Magic Screen.
Selecting the PUZZLE icon will bring up the Puzzle Map Screen.
Selecting the CHARACTER GOLD icon (bottom) will bring up the View Character
Screen.
