El último día 
 
 
Aquí están las instrucciones completas para el día final tal como se suministran con mi  
Disco original. 
   
 
Documentos redactados por John Panic " The Modem Man ". 
 
 
El objetivo de este juego es la terminación del inteligente RX-3000 
 computadora que ha tomado el control de la base EXPLORAN 1, planteando así un 
 tremendo peligro. 
La única manera posible de tener éxito reside en un código secreto que una vez introducido 
 en el sistema pausará el RX-3000 durante diez horas. 
 Desafortunadamente el código secreto está dividido en 13 partes que han sido 
 oculto en terminales de computadora en diferentes sectores de EXPLORAN 1. 
 Es su trabajo encontrar las piezas individuales, combinarlas de una manera lógica y  
 forma distinta y finalmente transmitir el código secreto a la CPU del 
 RX-3000. 
Durante el juego eres John, el hijo del profesor Warren. 
Cuidado: ¡EXLORAN 1 está equipado con los robots-guardias más peligrosos! 
¡¡Pero date prisa!!    Sólo te quedan 24 horas... 
 
EXPLORAN 1 se ha construido con cinco sectores: 
 
 1. Sector: Sector de almacenamiento y plataforma oriente 
 2. Sector: Sector del motor 
 3. Sector: Sector científico y plataforma oeste. 
 4. Sector: Sector laboral 
 5. Sector: Computadora principal 
  
Para encontrar las diferentes partes del código tendrás que comprobar cada 
 terminal único, ya que incluso una parte faltante completará el código 
 imposible. 
Al activar un terminal encontrarás el siguiente menú: 
 
 
 
     1."CÓDIGO DE RESOLVER" o "NO HAY CÓDIGO DISPONIBLE" 
     2."OPERACIÓN DEL DISCO" 
     3."ESTADO DEL JUEGO" 
     4."ENEMIGOS REALES" 
 
 
 
 
                          1. RESOLVER CÓDIGO 
 
 
Si la primera línea dice "SOLVE CODE" el terminal contiene una parte del  
 código secreto. 
Al elegir esta opción llegarás a un sistema especial que buscará 
 la parte oculta del código automáticamente. 
La parte no se mostrará directamente, sino que se le confrontará. 
 con una línea de figuras de las que falta la última figura. 
Por ejemplo: 1 2 3 4 5 ?? 
Las líneas siempre tienen una estructura lógica. 
Naturalmente, en el ejemplo anterior el número que falta es 6. 
Pero no siempre es tan fácil. 
Si ingresa la respuesta correcta, se liberará la parte deseada del código. 
 desde la terminal. 
Si el terminal no contiene una parte del código, la primera línea dice... 
 "NO HAY CÓDIGO DISPONIBLE". 
 
 
                           
                          2. OPERACIONES DEL DISCO 
 
 
Con esta opción podrás cargar y guardar hasta diez juegos diferentes. 
 configuraciones. 
Cada entrada muestra el nivel y el porcentaje de finalización cuando el juego 
 fue salvo. 
 
 
 
                          3. ESTADO DEL JUEGO 
  
 
Esta opción te permite ver todos los datos del juego actual: 
 
   - El porcentaje de finalización 
   - La lista de las partes encontradas del código. 
   - La cantidad de energía y oxígeno. 
 
 
 
                           
                          4. ENEMIGOS REALES 
 
 
Con esta opción podrás consultar la información técnica de todos 
 robots en este nivel. 
 
 
 
                               EXTRAS 
 
 
En algunos de los sectores encontrarás extras útiles que te facilitarán el juego.  
 el juego y de los cuales algunos incluso son necesarios para terminar el juego: 
  
 
     1. Tarjeta roja para abrir puertas 
     2. Tarjeta azul para salir del sector 
     3. Potencia de fuego 
     4. Tablero de control 
     5. Escudo 
     6. Máscara de oxígeno 
     7. Tarjeta magnética 
     8. Computadora portátil 
 
Para conocer el uso de estos extras ¡¡¡quedas solo!!!! 
Durante todo el juego tendrás que utilizar el joystick. 
A veces, sin embargo, también tendrás que utilizar el teclado.
