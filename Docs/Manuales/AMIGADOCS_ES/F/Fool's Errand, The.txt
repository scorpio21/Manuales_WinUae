Hacer el recado del tonto

  Este juego viene con una introducción animada llamada Prólogo.  esto
  se utiliza para presentar a los personajes principales de The Fool's Errand y comenzar
  la historia.  Está diseñado para ser visto antes de jugar el juego.
  Una segunda secuencia animada, llamada Finale, se puede ver en
  finalización del juego (pero no antes).  Sirve como recompensa por
  terminando el juego y representa la resolución de la historia en el juego.

INICIO DEL JUEGO: Cuando se te solicite el disco Workbench, inserta el
  Game Disk en la unidad interna.  El juego comenzará a cargarse y
  luego muestra la línea
       ¿Quieres ver el prólogo? (S/N).
  Escriba Y <enter> si desea ver el Prólogo, de N <enter> si desea
  Quiero ir directamente al juego.  Si elige ver el prólogo,
  se le pedirá que inserte el Show Disk y cuando aparezca el prólogo
  esté completo, se le pedirá que inserte el disco del juego y el juego
  comenzará.

     También puede optar por ver el Prólogo por separado insertando el
  Mostrar disco cuando se le solicite el disco de Workbench y luego desde el
  Indicador CLI, escriba
       Prólogo <entrar>
  y se mostrará el Prólogo.

VER EL FINAL: Una vez que hayas completado el juego, irás
  directamente al final.  Se le pedirá que inserte el disco Mostrar
  en cualquier unidad de disco y se mostrará el final.  si tienes
  has completado previamente el juego y deseas ver el final nuevamente,
  inserte el disco Show cuando se le solicite el disco Workbench.  Desde el
  Indicador CLI, escriba
       Final <entrar>
  y se mostrará el final animado.

  antes de jugar el juego.  El disco del juego siempre debe estar habilitado para escritura.
