COMANDOS DE TECLADO ADICIONALES
----------------------------
Si bien todas las selecciones del juego se pueden realizar con el mouse o el joystick,
Las siguientes "teclas rápidas" se pueden utilizar en varias partes del juego para
seleccione elementos, en lugar de usar el mouse/joystick para señalar y hacer clic.
Tenga en cuenta que la misma tecla controlará diferentes acciones en diferentes
circunstancias.

Los elementos que están sangrados debajo de otro elemento controlarán las acciones dentro
esa acción principal.  Como ejemplo, eche un vistazo al grupo de teclados.
comandos al lado del asterisco *.  En el garaje, al presionar 'N' aparecerá
hasta el periódico.  Con el periódico en pantalla, puedes presionar U o C
para ir a la sección de autos usados, A o P para ir a la sección de autopartes,
o G o ESC para volver al garaje.  Supongamos que presionó U o C
para ir a la sección de autos usados y la lista de autos usados aparece en la
pantalla.  Ahora, puede presionar 'N' para pasar a la página siguiente, 'P' para moverse
volver a la página anterior, o 'G' para volver al garaje.

Las teclas numéricas (1-9, 0, F1-F5) se utilizan para seleccionar elementos de un menú.
Por ejemplo, si el menú del oponente está en la pantalla del autocine, pulse
2 seleccionará el segundo oponente en la lista, la tecla F1 seleccionará el
11º oponente en la lista, etc. Los menús contendrán hasta 15 elementos a la vez.
en cualquier momento y se puede seleccionar con las teclas numéricas correspondientes:
1-9, 0, F1-F5 (se muestra a continuación como 1...F5).
