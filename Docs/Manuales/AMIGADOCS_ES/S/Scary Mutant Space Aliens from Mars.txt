SECCIÓN 1 - CONFIGURACIÓN

CREANDO UN DISCO DE GUARDAR
Para crear un disco guardado, primero debe formatear un disco que usará
para guardar su posición.  Nombra tu disco como 'Disco guardado SMSA'.

SECCIÓN 2 - CÓMO JUGAR

JUGAR EL JUEGO
Después de cargar el juego, verás la pantalla de reproducción de SMSA.  eso
consta de lo siguiente: un área de imagen, un cuadro de zona, un objeto
cuadro de imagen, el área de comando de clic rápido y el comando y texto
ventana.

EL ÁREA DE IMAGEN
Este cuadro contiene una imagen de su ubicación actual.  Dado que algunos de
Las habitaciones se parecen a otras, es posible que veas algunas de las fotos más.
más de una vez.  Pero no te preocupes... hay suficientes imágenes en el disco para
mantenerte entretenido.

LA CAJA DE ZONA
Este cuadro muestra una imagen en la que se encuentra actualmente, con el nombre
debajo de él.

LA CAJA SMSA
Aparece una imagen del último objeto móvil que ha examinado.
en esta caja.

EL ÁREA DE COMANDO DE CLIC RÁPIDO
Cada cuadrado aquí realiza el comando etiquetado.  Simplemente haga clic en el
casilla correspondiente.

LA VENTANA DE COMANDO
La ventana de comandos es donde aparecen los comandos que escribe.  esto es
También donde se muestran las descripciones de la habitación y los objetos cuando
entrar en una habitación o examinar un objeto.  Los comandos válidos se pueden dividir
en los siguientes cuatro grupos:

COMANDOS DE DIRECCIÓN
Los comandos de dirección le permiten moverse de un lugar a otro.
Estos comandos se basan en un sistema de brújula.  Los comandos válidos son
NORTE, SUR, ESTE, OESTE, NORESTE, NOROESTE, SURESTE y
SUDOESTE.
Ejemplos de comandos de dirección: OESTE, IR NORTE, NE

COMANDOS DE INVENTARIO
Para poder jugar debes recoger, usar y soltar objetos.
Para recoger un objeto, el comando es: TOMAR <objeto> donde está <objeto>
el nombre del objeto que deseas recoger.  Alternativamente puedes usar
las palabras: OBTENER, AGARRAR, RECOGER, OBTENER, ADQUIRIR, etc. También puede
TOMAR TODO o TOMAR TODO EXCEPTO/PERO <objeto>.

Para soltar un objeto, escriba: DROP <objeto> donde <objeto> es el nombre
del objeto que desea soltar.  Alternativamente, puedes usar las palabras:
DESCARTAR, DESECHAR, VOLCAR, DESECHAR, etc. También puede DESECHAR TODO o DESECHAR
TODOS EXCEPTO/PERO <objeto>.

Algunos ejemplos de objetos del juego son: CINTURÓN, TARJETA VERDE, QUESO,
TRANSMISOR.
Ejemplos de comandos de control de inventario: TOMAR LIBRO, TIRAR LA BOTELLA DE
VINO, OBTENGA TODO MENOS LA TARJETA VERDE, DEJE TODO MENOS EL HUEVO.

COMANDOS DE CONTROL DE OBJETOS

Es importante utilizarlos en el lugar y momento adecuados.  Un ejemplo de un
El lugar para usar un objeto sería una puerta cerrada.  debes desbloquearlo
antes de poder entrar.  Usted escribiría: DESBLOQUEAR LA PUERTA CON LLAVE.  No todos
Los objetos tienen usos en el juego.  Otros ejemplos de control de objetos

CUERDA AL INSECTO, CONSIGUE EL ROSQUILLO CON CUERDA.

OTROS COMANDOS
Al igual que los comandos de control de objetos, estos son para que usted los descubra y son
no directamente relacionados con objetos móviles.
Ejemplos de otros comandos incluyen: SALTAR A LA RAMA, PATEAR LA PARED,
ABRE LA PUERTA, GRITA, MUEVE EL ESCRITORIO, TOCA LA PARED.

SECCIÓN 3 - COMANDOS ESPECIALES
La siguiente es una lista de comandos y funciones especiales.

HISTORIAL DE COMANDO
Al presionar las TECLAS DE FLECHA ARRIBA o ABAJO se recorrerán los últimos 20
comandos que ingresó.  Una vez que aparece el comando, puede ser
editado antes de presionar RETURN.  La tecla DELETE elimina el carácter.
a la izquierda del cursor y al presionar RETROCESO se borrará el
línea de comando.

EXAMINAR
Utilice el comando examinar para obtener una descripción de la mayoría de las cosas.  tu puedes
examinar objetos, personas, puntos de referencia, etc. La sintaxis es: EXAMINAR
<objeto>.  Otros ejemplos incluyen: MIRAR DEBAJO DE LA CAMA, VER EN EL
CAJÓN.

AYUDA
Cada planeta tiene un párrafo de texto útil que le brinda sugerencias sobre
qué hacer en el planeta.  Para ver este texto, escriba AYUDA o presione el botón
Tecla AYUDA.

MAPAS INSTANTÁNEOS
Los mapas se pueden comprar durante el juego.  Una vez que tengas un mapa, puedes
véalo presionando F8.  Si tiene más de un mapa, presione F8 nuevamente
para ver el siguiente.  Presione ESC para salir.

INVENTARIO
Para comprobar lo que llevas, escribe: INVENTARIO o I o INV.

CARGAR
Para cargar un juego que haya guardado en su SMSA Save Disk, escriba:
CARGAR.
Luego, el programa te preguntará qué juego guardado (1-5) deseas cargar.

MACROS
Una macro es un comando definido por el usuario que se puede ejecutar con la tecla push.
de un botón.  Por ejemplo, en lugar de escribir ATACAR AL EXTRANJERO CON
THE BAT, puedes asignarlo a una tecla de función y simplemente presionar el botón
tecla para ejecutar el comando.  Las teclas de función f2, F3, F4 y F5 pueden
se le asignará una macro.  Para asignar un tipo de macro: Fn 'macro' donde n es
el número de tecla de función (2, 3, 4 o 5) y la macro es el comando que
quieres asignar.  Ejemplos: F3 'tomar todo', F4 'matar al monstruo',
F2 'examina la mesa con la lupa'.  Recuerda: usar
las macros, simplemente presione la tecla de función correspondiente.  para ver un
lista de sus macros actuales, escriba: MACROS.

Hay seis macros predefinidas: F1 - Salir, F6 - Cargar, F7 - Guardar, F8 -
Modo de mapa, F9: editar el último comando, F10: repetir el último comando.

LEER
Leer le brinda información sobre un objeto si hay algo escrito en él.
eso.  Para leer algo, escriba: LEER <objeto>.

REINICIAR
Para comenzar un juego desde el principio (y perder tu posición actual), escribe: REINICIAR.

HABITACIÓN
El comando ROOM le brinda información importante sobre la situación actual.
habitación, incluidos los objetos que se puedan llevar.  para conseguir una habitacion
descripción, tipo: HABITACIÓN o (L).  Tenga en cuenta que obtendrá un
descripción de la habitación cuando ingresa a una habitación.

GUARDAR
Para guardar su 'posición' actual en el disco, escriba: GUARDAR.  lo harás
luego se le solicitará el número del archivo guardado (1-5).

SALIR
Si quieres finalizar el juego, escribe: SALIR.  Es posible que desee guardar su
posición primero para que pueda continuar más tarde.

ESPERA
Hasta que escribas un comando, no pasa tiempo en el juego (excepto cuando
peleas).  Si quieres que pase el tiempo en el juego, escribe: ESPERA o (Z).
Por ejemplo, si estás esperando que un extraterrestre entre en la habitación para que
Puedes hacerle cosquillas hasta que cante la contraseña secreta, puedes conservarla.
al escribir ESPERA hasta que entre a la habitación.

SONIDO
Puede apagar el sonido ingresando: SONIDO APAGADO.  Para darle la vuelta
encendido, escriba: SONIDO ENCENDIDO.
