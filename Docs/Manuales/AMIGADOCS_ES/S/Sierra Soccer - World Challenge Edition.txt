REPETICIÓN DE ACCIÓN
-------------

Cuando una jugada que has creado o un gol que has marcado es tan espectacular
que desea verlo nuevamente, presionando cualquiera de las teclas F1 a F5, el
el juego se detiene y se te lleva a la función ACTION REPLAY, accediendo al
últimos seis segundos de juego.  Para REPRODUCIR la repetición presione la tecla F3, para PAUSAR
presione F2, para REBOBINAR presione F1, para AVANCE RÁPIDO presione F4 y para SALIR presione
F5.   También puede ajustar la visualización de la repetición utilizando cualquiera de los dos
palanca de mando.   Cuando ESTÁ EN PAUSA (F2), mover el joystick hacia la derecha acelerará el
reproducir a la velocidad de reproducción normal, lo que le permite ver la acción en
CÁMARA LENTA.   Al mover el joystick hacia la izquierda, el movimiento se ralentizará hasta que
comienza a retroceder.  Al presionar Fuego se sale de ACTION REPLAY.

CLAVES DEL JUEGO
------------

AMIGA IZQUIERDA = Formación y cambios de suplentes para Joystick 1

DERECHA AMIGA = Formación y cambios de suplentes para Joystick 2

F1 = Repetición de acción REBOBINAR

F2 = Acción Repetición PAUSA

F3 = Acción Repetición JUGAR

F4 = Repetición de acción AVANCE RÁPIDO

F5 = Acción Repetición SALIR

Barra espaciadora = Pausar/Reanudar juego (use el joystick para escanear cuando esté en pausa)

ESC = Salir del juego actual (un equipo humano perderá 5-0 si se usa)

S = Mostrar estadísticas del juego

C = Activar/desactivar multitud
