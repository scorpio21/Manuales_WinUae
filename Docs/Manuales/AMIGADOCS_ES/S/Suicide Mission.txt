- INSTRUCCIONES PARA LA MISIÓN SUICIDA -

Objetivo: bombardear todos los reactores en nueve niveles. 

                          - Controles del juego -

Solo desde la pantalla de título: M selecciona el control del mouse,
                            J selecciona el control del joystick (predeterminado),
                            El fuego inicia el juego.

Mientras juegas: P para pausar,
                            ESPACIO para reiniciar desde el modo de pausa,
                            Q para apagar el sonido,
                            S para activar el sonido,
                            Botón izquierdo o botón de disparo para disparar una bala.
                            B para lanzar un misil (cuando se usa el joystick),
                            Botón derecho para lanzar un misil (ratón)
                            ESC para volver a la pantalla de título.
                                                  
Para desactivar el ritmo del tambor de fondo, presione P cuando se alcance un nuevo nivel.
desplazándose por la pantalla.
Obtienes una vida extra cada 50000 puntos.
