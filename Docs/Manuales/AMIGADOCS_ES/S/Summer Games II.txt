                     JUEGOS DE VERANO II - MENÚ/INFORMACIÓN EVENTOS
                     ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
I) SALTO TRIPLE: presione FUEGO para iniciar la carrera. Luego toque DERECHA para omitir,
                      Luego toque IZQUIERDA para saltar.

J) REMO: presione FUEGO para comenzar, luego mueva de IZQUIERDA a DERECHA para acelerar
                      Arriba.
 
K) JAVELIN: toque FUEGO repetidamente para aumentar la velocidad de carrera, luego mantenga presionado
                      IZQUIERDA para comenzar a lanzar y soltar cuando llegue a su
                      Ángulo de trayectoria óptimo.

L) ECUESTRE: toque ARRIBA repetidamente para acelerar. Tire hacia abajo para girar
                      Alrededor/Trote hacia atrás. Presione FUEGO para volver a montar el caballo (saltar
                      está controlado por la computadora). 

M) SALTO DE ALTURA: ajuste la altura de la barra (1,20 M-2,50 M) con ARRIBA/ABAJO. Toque
                      DERECHO repetidamente para acelerar. Presione FUEGO cuando esté listo para
                      Saltar.

N) CERCADO: Mantenga presionado FUEGO y presione IZQUIERDA/DERECHA para moverse
                      Dirección. Para hacer una estocada, mantén presionado FUEGO y empuja ARRIBA/ABAJO.
                      Para controlar la espada, mueva el joystick en la posición deseada.
                      Dirección de la espada, sin FUEGO (Ej. ARRIBA-Mover espada
                      arriba, etc.).

O) CICLISMO: presione FUEGO y ARRIBA para comenzar a andar en bicicleta. Luego gire el
                      Joystick para acelerar.

P) KAYAK - Controles del joystick: paleta hacia adelante
                                                      |
                                   Girar canoa hacia la izquierda--+--Girar canoa hacia la derecha       
                                                      |
                                               Remar hacia atrás
