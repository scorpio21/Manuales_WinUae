EMPUJAR

 --------

                                      BLOQUEAR

 -PANTALLA DE OPCIONES-
 ---------------

 USANDO EL JOYSTICK O EL TECLADO, ESTO LE PERMITE:
 
 1) INGRESE UN NÚMERO DE CONTRASEÑA PARA OBTENER ACCESO
    A CUALQUIER NIVEL QUE HAYAS ALCANZADO ANTERIORMENTE.

 3) ESTA PANTALLA TAMBIÉN MUESTRA EL NÚMERO DE PAQUETES
    DE CORCHECAS RECOGIDAS HASTA EL MOMENTO (SI LAS HAY).
    

 -JUEGO-
 -----------

 Los objetivos del juego es manipular los bloques para lograr un 
 'reacción en cadena' de fichas de dominó que caen. La ÚLTIMA ficha de dominó en caer. Sin embargo,
 siempre debe ser el bloque TRIGGER. Todos los demás bloques, excepto el rojo
 TOPES, deben haber caído antes del bloque GATILLO.
 G.I.ANT puede recoger y transportar, y luego dejar caer, cualquiera de los bloques en
 Diferentes posiciones, excepto el GATILLO que no se puede mover.
 Muchos bloques tienen diferentes símbolos, lo que significa la propiedad particular.
 de ese bloque: en cualquier momento durante la reproducción, al presionar 'P' o F1 se pausará
 el juego y abre la pantalla de ayuda, mostrándote los símbolos
 y sus significados.
 
 Cuando esté seguro de haber colocado todos los bloques correctamente, podrá
 solo tiene UN empujón para tener éxito. Si tiene éxito, debe salir del campo abierto.
 puerta dentro de un límite de tiempo; de lo contrario, la pantalla tendrá que volver a reproducirse.
 Si no logra completar una pantalla, presione ESC para volver a intentarlo.
  
                           GIGANTE
 G.I.ANT es resistente y puede caer desde una altura bastante buena con solo un 
 cabeza aturdida. Sin embargo, hay un límite, y si G.I.ANT cae demasiado lejos,
 se desintegrará. El nivel se habrá reiniciado si esto sucede.
 Del mismo modo, puede quedar aplastado si queda atrapado bajo un bloque que se derrumba.
 También estoy muy feliz de subir y bajar escaleras con bloques, si es necesario.
 G.I.ANT es lo suficientemente fuerte como para transportar bloques a lo largo de la superficie, e incluso
 subir y bajar escaleras. Básicamente, lo único que es demasiado pesado para él 
 movimiento es el bloque GATILLO. Debido a la energía ejercida para manipular todos
 los bloques, G.I.ANT solo tiene fuerza suficiente para UN empujón por pantalla.

                           PAUSA/AYUDA
 Cuando se presiona 'P' o F1 durante el juego, esto PAUSARÁ el juego.
 y abre una pantalla de ayuda, que muestra y describe el 'ESPECIAL'
 bloques de dominó. Además, una vez que haya superado el límite de tiempo, esto 
 La pantalla también mostrará una PISTA sobre cómo completar la pantalla actual.

                     Qué hacer y qué no hacer
                     ------------


 ASEGÚRATE de que el GATILLO sea el último bloque en caer.
 NO permita que un bloque caiga directamente desde arriba sobre otro bloque,
 a menos que el bloque inferior sea un DIVISOR.
 MANTENGA la puerta de salida despejada.

                     FICHAS
                     ------

 Cuando completa una pantalla con éxito y dentro del límite de tiempo,
 se le otorgará un TOKEN. Los TOKEN se pueden utilizar de dos maneras:

 1) Si completa una pantalla, pero ha superado el límite de tiempo,
 se le permitirá utilizar un TOKEN para poder avanzar al
 siguiente nivel.

 2) si no pasa una pantalla por cualquier motivo que no sea exceder el límite de tiempo
 Puedes usar un TOKEN para reproducir la pantalla desde el punto en el que
 'empujado', es decir la pantalla se configurará como lo organizó anteriormente.NOTA-
 el cronómetro también se restablecerá a la hora en la que realizó el empujón.
                    
                     CORCHEBAS
                     -------

 En cada nivel, hay una pantalla que, cuando se completa con éxito, mostrará
 premiarte con un paquete de Corcheas. Se desconocen las pantallas específicas,
 pero necesitarás recolectar los diez paquetes para poder jugar la final
 pantalla. No recibirás tu paquete de Corcheas si has usado un
 token para completar esa pantalla en particular.

                     CONTRASEÑAS
                     ---------


 Después de completar cada pantalla, se le asignará un número, que
 te permite jugar el juego desde el último nivel en el que jugaste.
 El número de contraseña contiene información relacionada con:
 
 1) La pantalla a la que has progresado.
 
 2) La cantidad de paquetes de Corcheas recolectados
 El código de acceso NO contiene información relacionada con TOKENS, por lo que
 Cuando reinicies usando un código de acceso, no tendrás tokens.
 
                     CONSEJOS Y CONSEJOS
                     ------------


 1) Aunque solo tienes un PUSH, a veces se pueden lograr resultados 
 de otras maneras.
 
 2) No importa cuántos bloques haya en la pantalla, solo necesitas reorganizar uno
 A veces no es necesario mover bloques, solo es una cuestión de cuál
 ¡empujar!
3) Es posible (y a veces necesario) mover bloques en pie después
 ¡Ha comenzado una cascada!

 4) Siempre asegúrese de que G.I.ANT tenga acceso a la salida, aunque PUEDE moverse más rápido.
 que derribar bloques.

 5) si se te acaba el tiempo, siempre puedes acceder a la pantalla PAUSA/AYUDA 
 para una PISTA.
