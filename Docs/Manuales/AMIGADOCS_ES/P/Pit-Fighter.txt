 EL JUEGO:
 ----------

 Pitfighter: el deporte ilegal favorito del mundo, regresa para otro
 temporada, esta vez en más lugares y más popular que nunca.
 Únete a TY, KATO y BUZZ en una pelea sin límites contra ocho diferentes.
 oponentes, todos los cuales tienen sus propias técnicas y métodos de juego sucio.

 -----------------

 Intenta convertirte en el nuevo campeón mundial de Pitfighting. Tienes la opción

 TY - Kickboxer. Su mayor activo es la agilidad, mientras que sus movimientos especiales

 KATO - Cinturón Negro de Karate de 3er Grado. Su mayor baza es la velocidad. mientras

 Los movimientos son el golpe al cuerpo, el cabezazo y el martinete.

 |                                  |
 |                 SALTAR |
 |   MUDANZA ESPECIAL |   PATADA DE SALTO |
 |               \ |  / |
 |                \ | / |
 |   DEFENDER EL GOLPE-- --¡NADA!    |
 |                / | \ |
 |               / |  \ |
 |         RECOGER |   PATADA |
 |                 PATO |
 |                                  |
 +---------------------------------+

 Estos movimientos se aplican cuando estás mirando hacia la derecha. Los controles están invertidos. 
 cuando mira hacia la izquierda.

 EVADE, muy útil para salir de apuros.

 CONTROLES
 --------
 
 Jugador 1 - Joystick en el puerto 1
 Jugador 2: joystick en el puerto 0 (puerto del mouse)
 
 Pausa - H

 Reanudar cualquier tecla

 Dejar - Escapar

 Inicie un juego de 1 jugador/2 jugadores: siga las instrucciones en pantalla.
