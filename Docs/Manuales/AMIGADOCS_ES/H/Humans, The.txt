Controles del teclado
-----------------
PAD DE CONTROL = Teclas de cursor
Se utiliza para mover a tus humanos por el mundo.
LLAVE DE FUEGO
se utiliza para bloquear su elección de cualquier acción.
TECLA DE SELECCION = Barra espaciadora
Se utiliza para seleccionar una acción.
F1-F8
Se utiliza para seleccionar el humano que desea utilizar.
TECLA F10
Se utiliza para pausar y reanudar el juego.

Controles de palanca de mando
-----------------
ARRIBA = Mover humanos hacia arriba escaleras, pilas, etc.
ABAJO = Mover humanos hacia abajo por escaleras, pilas, etc.
       También recoger/soltar objeto.
IZQUIERDA = Mover humano a la izquierda
DERECHA = Mover derecho humano
BOTÓN DE DISPARO = Se utiliza para bloquear tu elección.
FUEGO+IZQUIERDA/DERECHA = Empujar roca
FUEGO al andar en roca = Saltar
FUEGO al llevar antorcha = Quemar
FUEGO+cuerda de transporte ABAJO = cuerda inferior
FUEGO+ARRIBA cuerda de transporte = Levantar cuerda

Controlando a los humanos
=======================
        Los humanos son tipos bastante fáciles de controlar (siempre que no
involucrarse demasiado emocionalmente).
        Al seleccionar un Humano, aparecerá una flecha grande momentáneamente.
encima de su cabeza, para ayudar a identificarlo.

Caminando a los humanos
------------------
        Para mover a un humano hacia la izquierda o hacia la derecha, presione el panel de control en la posición deseada.
dirección. Para mover a un humano hacia arriba o hacia abajo por una escalera o cuerda, presione el CONTROL
PAD hacia arriba o hacia abajo.

Cambiando a los humanos
---------------
        Para seleccionar un humano diferente, presione una tecla FUNCIÓN. El correspondiente
Entonces se seleccionará al humano.
        Alternativamente presione la tecla "<" o ">" para seleccionar el siguiente Humano en el
tribu. Este atajo puede resultar útil para la selección rápida de personajes.

Realizar una acción
--------------------
        En la parte inferior de la pantalla está la barra de acción. Aquí se muestra todo
de las acciones que un Humano puede realizar. Para desplazarse por los disponibles
acciones, presione la tecla SELECCIONAR. Cuando la acción deseada esté resaltada,
luego presione la tecla FUEGO para activar esa acción.

Usando la RUEDA
================
Uno de los mayores inventos del ser humano es la rueda. Este método de
El transporte se puede utilizar en varios puntos del juego.

Montaje de la rueda
------------------
        Para montar una rueda, mueva su Humano sobre la rueda que está en el suelo,
seleccione el ícono Recoger usando la tecla SELECCIONAR, luego presione la tecla FUEGO para bloquear
en tu elección.

Desmontar la rueda
---------------------
        Para desmontar una rueda, seleccione el icono Desmontar usando la tecla SELECT
luego presione la tecla FUEGO para bloquear su elección.

Montando la rueda
----------------
        Ahora estás al volante. Empuje el PAD DE CONTOL hacia la izquierda y hacia la derecha para
mueve la rueda. Observe que cuando baja una pendiente, la rueda acelera.
        Si pasas por el borde de una rampa, es posible que desees intentar saltar
usando la rueda. Para hacer esto, presione la tecla SELECCIONAR.
