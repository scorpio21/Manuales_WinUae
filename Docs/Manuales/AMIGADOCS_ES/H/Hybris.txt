AVISO: Para prevenir la infección viral del disco y también para tomarse el tiempo necesario para conectar el equipo necesario, 
apague su máquina antes de cargarla.  
Para jugar a HYBRIS, necesitarás un mouse (puerto 1) o un joystick (puerto 2). 
En cualquier momento durante el juego, 
puedes insertar el disco HYBRIS y presionar CONTROL A A (juntos) para iniciar automáticamente el juego.

PARA COMENZAR: Después de que el programa se haya cargado, aparecerá la pantalla de título.  
Elige tu papel como J.P. Maverick o como K. Lovett usando el ratón o el joystick 
para seleccionar la opción adecuada en la pantalla.  
Al presionar la barra espaciadora aparece el modo Cambiar opciones, 
que te permite cambiar los parámetros controlando el nivel de dificultad del juego.  
Para continuar, simplemente presiona la barra espaciadora nuevamente.  
El programa cambiará automáticamente al modo de demostración, mostrando el sistema de puntuación y la tabla de puntuación más alta.  
Presione el joystick o el botón del mouse para comenzar a jugar.
