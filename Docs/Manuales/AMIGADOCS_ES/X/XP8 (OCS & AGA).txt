    Instrucciones de carga:

    El disco 1 se inicia automáticamente: simplemente encienda su máquina y coloque el disco
    adentro. Verá algunas pantallas de título, luego la pantalla del menú principal de XP8
    (mantén pulsado el fuego para acelerar por las diferentes pantallas).  Disco 4
    Contiene un instalador de disco duro.

    La pantalla del menú principal:

    Desde la pantalla del menú principal tienes la opción de comenzar el nivel.
    indicado, (ver cómo jugar el juego) o para ingresar a las pantallas de opciones.  Usar
su método de control particular para resaltar las diversas opciones,
    presionando fuego para seleccionar.

    La pantalla de opciones principal:

    Desde aquí puedes cambiar varios aspectos del juego.  La mayoría son
    se explica por sí mismo.  Es desde esta pantalla que eliges uno o dos
    jugadores, activando o desactivando el selector. Asegúrate de tener tu
    método de control preferido seleccionado para cada jugador.

    Personalizar opciones:

    También puede ingresar a una pantalla de 'opción de personalización', desde aquí puede
    engaña tanto como quieras. Es posible hacer que el juego sea extremadamente
    fácil con todas las opciones activadas. Entonces, si quieres ver todo
    del juego, ¿quiénes somos nosotros para detenerte?

    Contraseña:

    Si ya has completado una misión y no deseas repetirla
    experiencia, puedes ingresar la contraseña proporcionada al final de la misión.

    Métodos de control para los jugadores 1 y 2:

    Joystick: Posiciones del joystick para movimiento.
    El botón de disparo dispara.
    Alt izquierda para alternar armas.
    (Alt derecha para el jugador 2).

    Joypad:
    Controlador direccional para movimiento.
    Botón rojo o verde para disparar.
    Botón azul o amarillo para alternar armas.
    Los botones laterales se desplazan hacia arriba y hacia abajo a través de las armas.
    Pausa/reproducir para pausar/reanudar
    Mientras está en pausa, presione Amarillo y Verde juntos para salir del nivel.

    Teclado: solo jugador 2
    Teclas de cursor para movimiento.
    Desplazamiento a la derecha para disparar o desactivar el disparo automático.
    Amiga adecuado para disparo automático.
    Alt derecho para alternar armas

    Controles adicionales: todos los métodos

    <Espacio> Naves morph
    < P > Pausar juego

    Mientras está en pausa:

    < Q > Abandona la misión
    < R > Reanudar misión
    <F10> Saltar al banco de trabajo
