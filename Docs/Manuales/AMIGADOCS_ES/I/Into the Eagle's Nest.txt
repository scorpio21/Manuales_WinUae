MENÚ DE CONTROL
------------

Presione la tecla CONTROL mientras está en la página de título para obtener el menú de control.  en
el menú Control use su Joystick para seleccionar la opción que desea usar y
use el botón Disparar para seleccionar la opción que necesita.

OPCIONES:
--------

DIFICULTAD (FÁCIL/DIFÍCIL): determina el número de golpes necesarios para matar a un
soldado enemigo (Uno o Dos).

CASTILLO (UNO/DOS): selecciona qué castillo deseas probar. El castillo dos es
más difícil de completar.

SONIDO (ON/OFF) - Enciende/apaga el sonido.

MENSAJES (ON/OFF) - Esto te permite desactivar los mensajes del juego.
INICIAR EL JUEGO: presiona el botón Disparar y luego selecciona la misión que necesitas usar.
el joystick (izquierda y derecha):

1 - Volar la fortaleza.

2 - Rescata a UN prisionero y haz volar la fortaleza.

3 - Rescata a DOS prisioneros y haz volar la fortaleza.

4 - Rescata a TRES prisioneros y haz volar la fortaleza.

Presione el botón de disparo para comenzar el juego.

Durante el juego, usa el joystick (en el puerto 2) para controlar a tu personaje o
Usa las teclas del cursor y la barra espaciadora para disparar.

Presione CONTROL para pausar

Pulsa la tecla ESC para cancelar el juego.
