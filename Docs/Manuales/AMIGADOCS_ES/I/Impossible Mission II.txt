instrucciones.  Lea esta tarjeta de comando para conocer los controles del teclado y
variaciones del manual para la versión Atari ST del juego.

Joystick: arriba, abajo, izquierda, derecha, botón de disparo.

Teclado: arriba - flecha arriba o I
           abajo - flecha abajo o K
           derecha - flecha derecha o L
           izquierda: flecha izquierda o J
           barra espaciadora o clr home (botón de disparo)

Teclado: arriba--8
           abajo--5 o 2
           derecha--6
           izquierda--4
           entrar (botón de disparo)

Claves adicionales:

*Presione 1 en el teclado para mover una plataforma deslizante hacia la izquierda.  Presione 3 en el
 teclado para moverlo hacia la derecha.
*Presione ESC para pausar.  Presione cualquier otra tecla para reanudar el juego.
*Presiona ALTERNAR S para guardar el juego.
*Presiona F10 para comenzar un nuevo juego.
*Presione ALTERNATIVA F10 para suicidarse (solo desde dentro de una habitación).
*Presione ALTERNATE Q para regresar al escritorio.
