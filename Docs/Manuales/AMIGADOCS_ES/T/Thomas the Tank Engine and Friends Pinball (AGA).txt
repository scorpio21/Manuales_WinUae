=========

Cuando hayas jugado un juego y hayas obtenido una puntuación alta, se te preguntará
para ingresar su nombre en la tabla de puntuación más alta. Hay una tabla de puntuación alta.
para cada mesa de Pinball.
Se le permite ingresar tres letras para su "nombre". Para hacer esto use el
teclas SHIFT izquierda y derecha para recorrer los caracteres alfanuméricos y
la barra SPCAE para establecer o arreglar un carácter. La mesa irá automáticamente.

cada vez que se sale de una mesa de Pinball y se vuelve a cargar el menú principal. por favor
tenga en cuenta que esta es la única vez que se guardan las puntuaciones altas.

Información general.
=====================

El JACKPOT es un bono de puntuación colectiva que acumulan todos los jugadores.
durante un juego. Se pone a cero cada vez que se otorga a un jugador.
