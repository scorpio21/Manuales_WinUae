Instrucciones de juego pequeño:


Controles del joystick: Izquierda: mueve el mosaico hacia la izquierda,
                     Derecha: mueve el mosaico hacia la derecha,
                     Abajo: acelera el mosaico hacia abajo,
                     Fuego: rotar la ficha.

Durante el juego, presione ESC para cancelar el juego. Para volver a Workbench presione ESC
durante el desplazamiento del título.

Pulsa 'P' para pausar el juego.


Los mosaicos especiales SÓLO se activan después de completar una línea con un especial
azulejo. Si no puedes completar una línea con un mosaico especial, se convertirá en
azulejo normal.

Azulejos especiales:


'Bom': cuando se activa, provocará una reacción en cadena y cada
Los espacios vacíos entre líneas se llenarán con bloques que caen.

'$ - Bonus': este mosaico agregará 10.000 puntos a tu puntuación cuando se active.


'+ plus': esto aumentará la velocidad del juego.

'- menos': esto reducirá la velocidad del juego.

'Pánico': cuando se activa, volteará la pantalla (o retrocederá hacia arriba).
          La puntuación será 3 veces mayor cuando esté activa. Durará 60 segundos.
          y luego volverá a su posición normal.

'N - Siguiente': esto activará o desactivará la pantalla de opción SIGUIENTE. Si la opción SIGUIENTE
             La pantalla está apagada, la puntuación será 2 veces mayor.

'R - Restablecer': el mosaico de reinicio se restablecerá:

              Velocidad del juego a la misma velocidad que al principio, según el
              nivel de imagen.

              El pánico se desactivará (si está activado).

              Se mostrará la siguiente pantalla de opción.

El nivel de dificultad se puede configurar en: FÁCIL, NORMAL (predeterminado), DIFÍCIL.
