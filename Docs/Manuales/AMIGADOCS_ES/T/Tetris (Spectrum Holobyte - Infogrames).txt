TETRIS para Amiga
                        --------------------

Cargando el programa

Encienda la computadora (los propietarios de Amiga 1000 deben usar Kickstart). Cuando se le solicite
para el disco Workbench, coloque el disco Tetris en la unidad. El juego será
"arranque automático".

Si está ejecutando CLI, coloque el disco Tetris en la unidad e inicie sesión en
conducir. Escribe tetris y presiona regresar. Desafortunadamente, el programa no puede
ejecutarse desde Workbench.

No existe una versión de accesorio de escritorio como se indica en la página 5 del manual. si
Si tienes 1 megabyte de RAM o más, Tetris puede ser multitarea.

Reglas generales para el juego

Su mouse tiene dos botones. Mantenga presionado el botón derecho para mostrar el menú.
barra. Mueva el puntero debajo de la opción que desee y luego "tire hacia abajo" la
menú. Arrastra el puntero para resaltar el elemento que deseas y suelta el mouse.
botón.

Utilice el botón izquierdo para seleccionar las opciones de altura y nivel.

Hacer una copia de seguridad

Tetris para Amiga está protegido por disco clave, lo que significa que puedes hacer
una copia de seguridad. La copia de seguridad no puede llamarse "Tetris". Debes insertar
el disco original, sin embargo, cuando se lo solicite el programa "Tetris".

Comandos del teclado

Varios de los elementos del menú Archivo se pueden seleccionar desde el teclado. Espera
presione la tecla Amiga (ubicada a la derecha de la barra espaciadora) y la
tecla adecuada simultáneamente.

                 Restablecer puntuaciones ........... Amiga - R
                 Torneo ............. Amiga - T
                 Abortar Juego ............. Amiga - A
                 Salir ................... Amiga - Q

Los elementos del menú Opciones también se pueden seleccionar desde el teclado.

                 Pausar el juego ............. P
                 Siguiente forma ............. N
                 Sonido ................. S
                 Sonido continuo ....... C
                 Ayuda ................... F1
                 Estadísticas ................. F2
                 Nivel ................. F3
                 Descripción ............ F4
                 Modo avanzado .......... A

Otro

El modo avanzado se selecciona desde el menú Opciones o el teclado, no desde
la pantalla de nivel/altura como en la versión para Mac.

No puedes arrastrar los paneles en la pantalla de Amiga.
