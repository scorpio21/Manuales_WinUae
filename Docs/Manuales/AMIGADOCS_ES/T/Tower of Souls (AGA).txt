LA BARRA ESPACIADORA activa o desactiva el 'joystick virtual'.
El joystick virtual está activado si el puntero es una luz guía.
El joystick virtual está desactivado si el puntero es una mano.

Mientras el joystick virtual está activado:

IZQUIERDA - Usa el arma actual

DERECHA: lanzar el hechizo actual

Mientras el joystick virtual está apagado:

IZQUIERDA: activa el icono requerido, EXCEPTO cuando se sostiene comida o
beber. (Consulte la sección sobre Pantalla de estado y Uso de la barra de menú).

DERECHA: activa funciones alternativas en la barra de menú.
Vuelve a encender el 'joystick virtual' si no hay alternativa
función encontrada.

usando un hechizo de sirviente invisible).   Presione ESPACIO para volverse virtual
