INSTRUCCIONES DE CARGA DEL DISCO
Usuarios de Amiga 500
   Enciende la computadora. Cuando aparezca el símbolo del banco de trabajo, inserte el
   disco del juego en la unidad interna. El juego se cargará automáticamente.
Amiga 1000 usuarios
   Enciende la computadora. Inserte su disco Kickstart. Cuando el banco de trabajo
   Aparece el símbolo, inserte el disco del juego en la unidad interna. el juego
   se cargará automáticamente.
CONTROLES
  Este juego se juega con un joystick.
  *IMPORTANTE* Conecte el joystick al puerto 2 antes de encenderlo.
Después de que el juego se haya cargado:
   Pulsa el botón FUEGO para mostrar la pantalla de selección de jugador.
   Mueva el joystick hacia arriba o hacia abajo para seleccionar un juego de UN jugador, DOS jugadores
   juego, o un juego de PRÁCTICA PARA UN jugador.
   Presione el botón FUEGO para comenzar a jugar.
Durante el juego:
   Mueva su marcador hacia arriba/abajo/izquierda/derecha con el joystick.
   Mantenga presionado el botón FUEGO para comenzar un sorteo LENTO.
   Al soltar el botón FUEGO se inicia un sorteo RÁPIDO.
   Presione F1 para pausar el juego; presione F1 para reanudar la reproducción.
