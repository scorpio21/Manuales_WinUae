Documentos para
                            Operación Sprauce
                            ------------------

Viendo desde: La plataforma de control
------------------------------
R -> Invierte los motores de los barcos.
F -> Los motores del barco giran hacia adelante
P -> Potencia máxima (100% de velocidad)

0-9 -> Velocidad (0 más lento / 9 más rápido)

F-10 -> Vista trasera derecha
F-9 -> Vista lateral derecha
F-8 -> Vista frontal
F-7 -> Vista lateral izquierda
F-6 -> Vista trasera izquierda
F-3 -> Radares
F-2 -> Monitor de puente
F-1 -> Visualización del área táctica


De: El Monitor del Puente
-----------------------
N -> Estrella de navegación
C -> Constar

G -> Lanzar paja
F -> Lanzar bengala
D -> Cargar señuelo / seguido de F-5 para lanzar señuelo

F-4 -> Informe de daños


De: Constar
-------------
Constar -> Esto activará tu satélite estratégico para localizar
           objetivos enemigos y brindarle información sobre:
           Identificación / Velocidad / Rumbo / Altitud


De: Navstar
-------------
1 -> Mapa de vigilancia: mostrará todo el golfo y mostrará
                        tú el clima / barcos / aviones
                        submarinos/bases y su posición actual.
                        

2 -> Mapa de profundidad regional -> muestra su posición actual y muestra
                           las profundidades de las aguas que te rodean.


De: La exhibición del área táctica
-------------------------------
barra espaciadora -> Cambia el bloqueo de un barco a otro.
B, N y M -> Funciones de zoom del mapa (B - acercar / N - centrar / M - alejar)

Las siguientes teclas se pueden usar una vez que haya un bloqueo (barra espaciadora) en su
objetivo:

C -> Visualización de bloqueo de imagen de computadora (presione F-1 para salir)
T -> Bloqueo táctico

H -> Arpón de fuego
G -> Dispara armas de 45 mil
S -> Dispara misiles de gorrión
R -> Dispara Asrocs
