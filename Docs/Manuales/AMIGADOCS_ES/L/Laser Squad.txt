INTRODUCIÓN

Laser Squad es un sistema de juego de guerra táctico para uno o dos jugadores que
Simula con precisión muchos aspectos del combate a nivel individual.
tomar algo de tiempo para que un nuevo jugador se familiarice con todos los aspectos de
el sistema, y le recomendamos que cargue el programa y siga
la primera sección de este manual que le proporciona una guía básica
para jugar el juego. La segunda sección debe ser tratada como una
Manual de referencia para jugadores experimentados. Algunas características de la
El sistema no se implementa con los cinco escenarios que proporciona la
programa principal, pero los escenarios futuros explorarán la versatilidad de
juego que Laser Squad puede proporcionar. La sección final de este
El folleto proporciona instrucciones para los cinco escenarios proporcionados con el
programa principal.

RESUMEN

Laser Squad se juega en una serie de turnos de juego. Cada jugador, durante
En su turno, controla un pequeño número de unidades que representan a humanos,
droides u otras criaturas. Cada unidad tiene su propio nombre individual y
características que afectan las capacidades y la actividad de esa unidad.
Al comienzo de cada turno de juego, a cada unidad se le asigna un número de
Puntos de acción (o AP) que se utilizan para realizar todas las actividades de
esa unidad, como movimiento, combate, manipulación de objetos, etc. Una vez que todo
Se utilizan puntos de acción para cada unidad, o el jugador no desea
usar más, se selecciona la opción 'finalizar turno' y el otro jugador
(o la computadora) toma su turno. El ganador del juego es el primero.
lado para acumular 100 puntos de victoria. Los puntos de victoria se otorgan por
varias cosas, como eliminar unidades enemigas o destruir ciertas
instalaciones, dependiendo del escenario que se esté jugando.

MOVIMIENTO

Mover una unidad siempre costará Puntos de Acción, dependiendo del terreno.
eso está siendo movido. Moverse en diagonal costará dos AP adicionales
y rotar una unidad 45 cuesta un AP. Si intenta mudarse a
terreno intransitable la unidad atacará el terreno y posiblemente
destruirlo.

Puedes mover la unidad seleccionada sobre unidades amigas, pero
nunca puede y movimiento encima de una unidad amiga. Las únicas opciones
disponibles para una unidad encima de otra son continuar moviéndose o
seleccione la opción FINALIZAR MOVER del menú. Si selecciona FINALIZAR MOVER
opción entonces la unidad se colocará en el último espacio vacío que
ocupado.

costo, el daño y la ubicación del impacto.

LÍNEA DE VISIÓN

Laser Squad utiliza un sistema de movimiento oculto, lo que significa que las unidades enemigas
no aparecerán en el mapa a menos que estén en el campo de visión de
una de tus unidades. El campo de visión de una unidad se extiende 45 desde el frente.
Varios tipos de terreno, como paredes o puertas, bloquearán la línea de
vista. No verás una unidad enemiga hasta que esté claramente a la vista.
