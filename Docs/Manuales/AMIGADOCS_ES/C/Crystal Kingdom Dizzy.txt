GENIO DEL JUEGO
Game Genie es un personaje al que le encanta ayudar a la gente a jugar.  si tu 
Dale algunas palabras clave especiales y él hará todo lo posible para ayudarte.
Anota los códigos especiales que se te entregan al final de cada acto.  en el
comienzo del juego, puedes regresar instantáneamente a este acto caminando 
a través de las puertas marcadas como acto 2, 3 o 4. En este punto, Game Genie
Le preguntará su palabra clave, escríbala usando el teclado.  Si no lo sabes
cualquier código simplemente salga del lado izquierdo de la pantalla.


CONSEJOS Y CONSEJOS
* Recoge los objetos
* Game Genie puede ayudarte
* Es posible que los piratas hayan robado el tesoro.
