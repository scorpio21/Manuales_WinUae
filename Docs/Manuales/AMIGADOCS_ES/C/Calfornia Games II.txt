INFORMACIÓN DE EVENTOS:
~~~~~~~~~~~~
Ala delta: Empuje hacia arriba para escalar (volar hacia arriba), tirar hacia abajo para sumergirse (volar hacia abajo). prensa
             Fuego para lanzar bombas.

SNOWBOARDER: use controles normales para el helicóptero, presione fuego para soltar
             Jugador.
             EJECUCIÓN PRINCIPAL: muévase hacia la izquierda y hacia la derecha para controlar al jugador. Presione fuego para
             Saltar.
             SNOW BOWL: muévete hacia la izquierda y hacia la derecha para controlar al jugador. Presione fuego para
             Planta de mano.

BODYBOARDER: (Sin presionar Fuego) Muévase hacia la izquierda para subir la ola, muévase hacia la derecha para
             Bajar ola. (Con Fuego presionado) Mover hacia la izquierda para rotar el tablero
             Derecha, muévase hacia la derecha para girar el tablero hacia la izquierda.
             SEGUNDA PARTE: Muévete hacia arriba y hacia abajo para evitar los objetos que se aproximan.

PATINADOR: Muévase hacia la izquierda y hacia la derecha para girar hacia la izquierda y hacia la derecha. Pulsa Fuego a Ollie.

JET SURFER: muévete hacia la izquierda y hacia la derecha para controlar al jugador. Mantén el fuego para acelerar.

INFORMACIÓN DE LLAVES:
~~~~~~~~~~
Utilice las teclas del cursor igual que el joystick. Espacio/Retorno - Fuego.
F1 - Sonido activado/desactivado. F2 - Pausa/Reanudación de pausa. F10 - Modo PAL. ESC - Salir del evento.
