PANTALLA DE TÍTULO

	Haga clic en el botón izquierdo del mouse para avanzar a la siguiente pantalla.

PANTALLAS DE OPCIONES y AYUDA

	Utilice el ratón para señalar una opción y luego haga clic con el botón izquierdo del ratón.

EL GRAN SALÓN

	Seleccione una puerta para que entre el héroe escribiendo un número del "1" al "4", o
use el moust para señalar una puerta y luego haga clic con el botón izquierdo del mouse para elegir
eso. 

	Presione la tecla ESC para regresar a la pantalla Opciones.

PANTALLAS DE JUEGO

Teclado y ratón

	El teclado se utiliza para controlar los movimientos del héroe.

Presiona la tecla "A" para mover al héroe hacia la izquierda.
Presiona la tecla "D" para mover al héroe hacia la derecha.
Presiona la tecla "W" para hacer que el héroe suba.
Presiona la tecla "S" para hacer que el héroe caiga.
Presiona la tecla "E" para hacer que el héroe se agache y la barra espaciadora para hacerlo saltar.
La tecla "Q" es la tecla de Acción, que se utiliza para recoger objetos y activar cosas.

Presiona la tecla TAB para pausar el juego y luego cualquier tecla para continuar. 
Presione la tecla ESC para regresar a la pantalla Opciones.

Para lanzar piedras, utiliza el ratón para colocar la zona del héroe y luego haz clic en el
botón izquierdo del ratón.

Palanca de mando:

	Mueva el joystick hacia la izquierda y hacia la derecha sin presionar el botón para moverse
el Héroe a izquierda y derecha. Mover el joystick hacia arriba hará que el héroe salte o
moverse "hacia arriba" (por ejemplo, subir escaleras o subir escaleras), dependiendo de su posición.
	De manera similar, mover el joystick hacia abajo hará que el héroe se agache o
moverse "hacia abajo".

	Para lanzar piedras, mantén pulsado el botón y mueve el joystick hacia arriba o hacia abajo.
para posicionar el arem del héroe. Con el botón aún presionado, mueva el joystick
hacia la izquierda o hacia la derecha para que el héroe arroje una piedra en esa dirección.

	Presione el botón sin mover el joystick para recoger objetos y
activar cosas. Mover el joystick en dirección diagonal hacia arriba hará que el
Héroe para dar un salto corriendo.
