		     EL MANUAL (alemán) DE GIGANOID


INICIO:

AMIGA 500/AMIGA 2000
Enciende tu Amiga y luego inserta el disco GIGANOID en él.
Conduzca DF0. El programa se carga y se inicia automáticamente.

Amiga 1000
Enciende tu Amiga e inserta el Kickstart. despues del barco
Puede insertar el disco GIGANOID directamente. El programa se carga y
comienza automáticamente.

MENÚ GIGANOIDE
Después de cargar, aparecerá presionar el mouse o el joystick una vez
botón el menú GIGANOID.

Puedes configurar el modo de juego:
F1 - número de jugadores
F2/F3: asignación de puertos y control del mouse o joystick
yo - información
historia s
C - programador

Una vez que haya configurado todo, puede presionar el botón 
Botón del mouse o joystick para iniciar el juego.

el juego

La idea del juego es que intentes golpear la pelota así con tu raqueta.
para dirigir los bloques para que desaparezcan. Después de conocer cierta MAGIA
bloques, se lanza una bola mágica voladora contra ti. Prueba esto
para atrapar. Si lo logras, se activarán las siguientes propiedades:

Letra de bola mágica:
C - Detiene el balón
L-láser
D - 2 bolas
B - Siguiente nivel
E - el tablero se hace más ancho
S - La bola se ralentiza
P - 1 vida gratis

¡Tienes que sobrevivir a 50 cámaras para convertirte en GIGANOID!
Una tarea difícil, porque además hay:

Los DEMONIOS:
En esta ronda de bonificación tienes la opción de saltarte las cámaras. Ge-
Especifique cuántas cámaras desea omitir. Todavía hay un problema.
Cuantas más cámaras quieras saltar, más rápido dispararán los demonios.
con balas mágicas hacia ti.

EL PORTERO/EL GRAN MAESTRO:
Se les dispara con bolas brillantes. Intenta usar las cabezas de demonio.
golpear la pelota con la mayor frecuencia posible. Después de un cierto número de visitas
el demonio admite la derrota.

Puntuación alta
Después de un juego exitoso, puedes encontrarte en la lista de los mejores resultados.
Estado de inscripción. Mueve el ratón o mueve el joystick
para avanzar las letras. Presione Fuego o el botón izquierdo del mouse
para arreglar la carta.
