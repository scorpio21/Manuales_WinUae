COMENZANDO EL JUEGO

Para comenzar a jugar VORTEX debes presionar una de las siguientes teclas:

F1: modo para un jugador.

F2: modo para dos jugadores.

F3 - Modo equipo (se necesita un segundo ratón).

F4 - Modo guerra (se necesita un segundo ratón).

El modo para un jugador permite la acción de un jugador.

El modo de dos jugadores permite que dos jugadores jueguen alternativamente VORTEX y comparen
sus logros.

El modo Equipo permite que dos jugadores jueguen de forma cooperativa compartiendo recursos.

El modo Guerra permite que dos jugadores se desafíen entre sí y también al VORTEX.
