BLOCKBUSTER de Mindscape 
 
PARA EMPEZAR 
 
1. Debes usar el mouse solo en el puerto 0. 
2. Inserte el disco Blockbuster, con la etiqueta hacia arriba, en la unidad de disco. 
3. Encienda el monitor o televisor y la computadora. 
4. Presione el botón izquierdo del mouse para iniciar el juego.

CONTROL DEL JUEGO 
 
Usa el ratón para controlar el murciélago.   Presione el botón izquierdo del mouse para iniciar  
la pelota y usar armas.   Presione el botón derecho del mouse para seleccionar y  
implementar armas.

PUNTUACIÓN Y BONIFICACIONES 
 
El color de un ladrillo determina el número de puntos obtenidos (de 5 a 75)  
cuando es destruido.   Los extraterrestres valen 100 puntos.   Al final de un  
pantalla completada con éxito, los tokens no gastados valen 1000 puntos cada uno,  
hasta un máximo de 9000 puntos. 
 
Se otorgan bolas extra por cada 50.000 puntos obtenidos. 
 
En algunas pantallas puedes ganar puntos de bonificación destruyendo, EN ORDEN, ladrillos.  
marcado con las letras B-O-N-U-S.

DISEÑANDO TUS PROPIAS PANTALLAS 
 
Hay 48 pantallas definibles por el usuario (81-128).   
 
1. Acceda al Diseñador de pantalla manteniendo presionada la tecla RETORNO mientras el  
el juego se está cargando. 
 
2. Para acceder a la pantalla de ayuda integrada, presione la tecla AYUDA. 
 
3. Utilice las 4 teclas de flecha para mover el cursor por la pantalla y presione  
la barra espaciadora para colocar un ladrillo en la pantalla. 
 
4. Utilice las teclas - o + para recorrer los colores disponibles.  para cambiar un  
color de un ladrillo existente, coloque el cursor sobre el ladrillo y seleccione un  
color diferente. 
 
5. Para borrar un ladrillo, coloque el cursor sobre el ladrillo y cambie el  
color a negro. 
 
6. Después de colocar un ladrillo, agregue características especiales (como Reflejo láser  
o puntos de bonificación) a los ladrillos presionando la tecla correspondiente (enumerada  
a continuación). 
 
7. Para establecer el nivel de dificultad de la pantalla, presione RETORNO para ir a la  
Pantalla de parámetros.   Escriba el valor del parámetro cuando se le solicite.   uso  
las teclas de flecha arriba y abajo para recorrer los parámetros.   Presione VOLVER  
cuando haya terminado de configurar los parámetros. 
 
8. Presione CLR/HOME para diseñar la siguiente pantalla.   Presione INSERTAR para regresar a  
la pantalla anterior. 
 
9. Presione F1 para guardar las pantallas que ha diseñado en el disco del programa. 
 
10. Después de haber guardado sus pantallas, reinicie la computadora y cargue el  
juego.   Para cargar sus pantallas, escriba la contraseña USUARIO en la pantalla de título.   
Luego, el juego comienza en la pantalla 81.

Después de haber colocado un ladrillo, presione cualquiera de las siguientes teclas para agregar el  
características que deseas.   Un ladrillo puede tener más de una característica, pero no puede ser  
tanto multigolpe como indestructible. 
 
     M Ladrillos multigolpe 
     I Ladrillos indestructibles 
     R Reflejar láseres 
     B,O,N,U,S Ladrillos de bonificación (puedes usar todas las letras para obtener puntos de bonificación)
