Línea de fuego
------------

-> truco #1 <-
 Antes de que comience el juego, presiona la barra espaciadora y escribe 'OPERACIÓN HURÓN'. La pantalla
 Ahora debería cambiar y serás totalmente invulnerable con la capacidad de 
 saltar niveles presionando '0' a '9'.
-> truco #2 <-
 Cuando el juego se cargue, presiona AYUDA para recibir un mensaje secreto y luego escribe
 'QUÉ LADO' para créditos infinitos.  Usa las teclas numéricas para saltar niveles.
