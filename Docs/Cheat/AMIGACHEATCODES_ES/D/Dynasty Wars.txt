Guerras de dinastía
------------

-> truco #1 <-
 Presione <F9> para hacer una pausa, luego presione SHIFT, HELP, '1' y <F10> simultáneamente,
 seguido de reanudar el juego con <F10>. Ahora <F2> te permite saltar niveles.

-> truco #2 <-
 Cuando aparezca la pantalla de título, escribe 'MODO TRUCOS'. De ahora en adelante, presione <F2> para
 pasar al siguiente nivel.
