Rick peligroso 2


 MODO TRUCO:

 Como en el primer juego, ingresa 'POOKY' en la tabla de puntuación más alta.
 Ahora, la próxima vez que presiones fuego para iniciar el juego, tendrás la opción
 para seleccionar y jugar una versión de 8 bits o una versión de 16 bits del juego.
 Simplemente mueve el joystick hacia la izquierda o hacia la derecha para elegir.

 También intenta escribir 'BURN IN HELL' en la pantalla de título para obtener vidas ilimitadas.
