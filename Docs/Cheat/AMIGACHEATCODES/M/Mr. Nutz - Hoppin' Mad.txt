Mr Nutz


 CHEAT MODE:

 On the map screen, type 'DONTPANIC' to enable the following keys:

   [F2] - Level 2.
   [F3] - Level 3.
   [F4] - Level 4.
   [F5] - Increase lives.
   [F6] - Decrease lives.
   [F7] - Increase gems.
   [F8] - Decrease gems.
   [F9] - Increase bombs.
  [F10] - Decrease bombs.
 [Help] - Fly mode on.
  [Del] - Fly mode off.

 Pressing the function keys during the platform game activates the
 various power-up's and sheilds.


 TIPS:

 If you collect 99 gems you will be given a star which allows you
 to save your game and to enter bonus levels.
