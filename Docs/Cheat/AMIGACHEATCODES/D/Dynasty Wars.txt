Dynasty Wars
------------

-> cheat #1 <-
 Press <F9> to pause, then press SHIFT, HELP, '1' and <F10> simultaneously,
 followed by unpausing the game with <F10>. Now <F2> lets you skip levels.

-> cheat #2 <-
 When the title screen appears, type 'CHEAT MODE'. From now on, press <F2> to
 skip to the next level.
