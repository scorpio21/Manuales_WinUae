 CHEAT MODE:

 On the option/title screen, type 'ELV or 'CHAMP'. Now press the [Help] key
 followed by the fire button for instructions on using the cheat mode.

 Cheat Keys:

   [F1] - Scatterbolts.
   [F2] - Multi fire.
   [F3] - Auto repeat.
   [F4] - Armour missiles.
   [F5] - Stun.
   [F6] - Neutron bomb.
   [F7] - Clone ship.
   [F8] - Shield.
   [F9] - Skip level.
  [F10] - Extra power bars.
 [Help] - Show list.

 If you are playing a two player game hold down 'DEL' and the desired
 function key to affect the second player.
 
 If you type in 'ELVIE' or 'CHAMPIE', the game reports that the super cheat mode
 has been entered. .  You will now be able to use the following keys 
 on the title screen to change the games parameters:

   [F1] - New drain value    
   [F2] - New rapid drain value    
   [F3] - New machine drain value    
   [F4] - New delay value    
   [F5] - New rapid delay value    
   [F6] - New machine delay value
   [F7] - New gain per frame
   [F8] - New max ship speed
   [F9] - Press a number 1-5
  [F10] - switch between screens
