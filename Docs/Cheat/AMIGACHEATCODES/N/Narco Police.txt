Narco Police


 CHEAT MODE:

 Type any of the following on a computer or during play:

    'ABRIR' - Open all security doors.
    'BLAST' - For a smart bomb.
 'MUNICION' - 200 ammunition.
 'NOENEMIG' - Remove enemies (possibly 'NOENEMIS')
 'COMENZAR' - Replace enemies.
 'ETAPAUNO' - Moves you one zone.
 'NOAMETZ#' - Turns off guns in section # (replace # with 1 - 4).
  'NOCAMZ#' - Turns off cameras in section # (replace # with 1 - 4).
 'CUADRICU' - ?
 'ETAPADOS' - ?
   'CONGRA' - Finish the game.
