Cool Spot


 CHEAT MODE:

 Pause the game and using the joystick create the word COOL as follows.
 Note that you have to center the joystick after every move.

 Left, Down, Right, Fire     - C
 Left, Down, Right, Up, Fire - O
 Left, Down, Right, Up, Fire - O
 Down, Right, Fire           - L
 Up, Down, Left, Right, Fire - Return

 Fire enables each letter. When you hear cool spot say "Yeah",
 unpausing the game at that point will complete the level. Also when
 you here cool spot say "OK", unpausing at this point will complete
 the level and take you to the bonus game. However, if you complete
 all the above movements in one go then the game will automaticly
 restart and the following keys are activated:

 [1] to [0] - Jump to level 1 to 10.
      [F10] - Next level.
       [F9] - Return to level 1.
       [F3] - Gives 6 continues.
        [Z] - Move spot around screen (use two keys below).
        [+] - Speed spot up.
        [-] - Slow spot down.
