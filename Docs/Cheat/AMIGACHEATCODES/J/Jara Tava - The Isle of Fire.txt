JARA-TAVA - The Isle of Fire

(You begin on a wide, sandy beach) N, GET PARACHUTE, N, E, E, SWIM (to 
the ship), D, GET ROPE, U, SWIM, 3xW, S, READ MESSAGE (a hint on where 
to find the treasure), N, ENTER HUT, GET NOTE, READ NOTE (how to build 
a glider), DROP NOTE, W, N, N (this is a good spot for building the 
glider), DROP PARACHUTE, S, W, TIE ROPE (to a tree), D, ENTER CAVE, GET 
CUTLASS, W, U, DROP MAP, GET ROPE, E, N CUT BAMBOO (with the cutlass), 
DROP CUTLASS, GET BAMBOO, N, BUILD GLIDER, GET GLIDER, FLY (you fly to 
Jara-Tava)

D, READ SIGN, DROP GLIDER, W, N, N, U, N, ENTER PLANE, GET RAT, S, S, 
D, S, E, E, PUSH TREE (it forms a bridge), N, THROW RAT (the snake 
slithers off), U, GET SHOVEL, D, S, W, W (here are the orchids 
mentioned in the message), DIG (you have found the treasure!!)

Notes: 
1. On Jara-Tava, you have a limited number of moves before the volcano 
erupts. However, this number is rather large, so you should have plenty 
of time.
2. The crocodile seems to be harmless 
3. The strange fruit is useless.
