Voyager


 CHEAT MODE:

 Typing 'WHEN THE SWEET SHOWERS OF APRIL FALL' on the option screen
 gives you access to the cheat menu. It also enables the following keys
 during play:

  [F1] - Go down a level.
  [F2] - Go up a level.
  [F3] - Cycle through languages.
  [F8] - Game Position.
  [F9] - Data on object.
  [F10] - Frame rate.

 Pressing [Enter] on the main option screen gives these extra options:

  [Shift] - Cycle through objects.
    [Del] - Cycle through objects.
   [Help] - Cycle through objects.
 [Cursor] - Rotate object.
      [7] - Decrease size of object.
      [4] - Enlarge size of object.

 Check out object number 0058.
