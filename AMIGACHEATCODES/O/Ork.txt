Ork


 LEVEL CODES:

 The code to gain access to level three is: 2314
 Use the joystick to enter the code at the end of level two.


 CHEAT MODE:

 Stand by a computer terminal and pull down and press fire.
 Now point the cursor in all four corners of the options screen.
 The cheat mode will now be active. This enables these keys:

      [A] - Extra ammo.
      [F] - Extra fuel.
      [H] - Health top up.
 [Return] - Toggle jet pack on/off.
