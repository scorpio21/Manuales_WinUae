James Pond 3 (Operation Starfish)


 CHEAT MODE:

 During any level in the game type 'NIGHTMARE', then press [F10].
 You will be given a cheat menu to use as you will.
 Once you've selected various options in the cheat menu just
 press the [Esc] key to exit and return to the game.

 You can also try typing in any of the following on the map screen:

        'EVAS' - Save game at any time.
      'FORMAT' - Erase all previously saved games.
 'UNCLE ROGER' - Message from the programmer.
